## 🧭 What this app does

laravel-ai-aegis is a local-first security layer for the Laravel AI SDK. It helps protect AI prompts before they reach your model.

It does two main jobs:

- Blocks prompt injection attempts
- Replaces personal data with safe placeholders before sending text to the LLM
- Restores the original values in the response

This helps keep user data private while reducing the risk of unsafe prompt content.

The app checks prompt content for unsafe instructions that try to change how the model behaves.

### PII pseudonymization
The app can replace personal data with safe labels before the prompt leaves your machine.

### Local-first flow
It keeps the security step on your side first, which gives you more control over the data path.

### Response restoration
After the model returns a response, the app can map protected values back to the original text.

### Laravel AI SDK friendly
The design fits Laravel AI SDK use cases, so it can sit between your app and the model layer.

## ⚙️ How it works in simple terms

Think of it as a filter between your app and the AI model.

1. Your app sends a prompt
2. laravel-ai-aegis checks the text
3. It flags or blocks unsafe instructions
4. It swaps personal data for safe placeholders
5. The model gets the cleaned prompt
6. The response comes back
7. The app restores the original values

This keeps sensitive data out of the model input while still letting your app use the response.
